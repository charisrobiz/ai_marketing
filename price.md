## 7. 외부 서비스 API 비용 총정리

### A. 실연동 완료 (API 호출 코드 구현됨)

데이터 조회(READ)만 수행하며, **API 호출 자체에는 토큰 비용이 발생하지 않습니다**.

| 플랫폼 | 용도 | API 호출 비용 | 실제 서비스 비용 |
|--------|------|-------------|----------------|
| **Meta Ads API** | 캠페인 성과 조회 (노출, 클릭, 지출) | 무료 | 광고 집행 비용 별도 (CPC $0.5~$3+) |
| **Google Ads API** | 캠페인 성과 조회 (노출, 클릭, 비용) | 무료 | 광고 집행 비용 별도 (CPC $1~$5+) |
| **AppsFlyer API** | 설치 어트리뷰션 데이터 조회 | 요금제별 상이 | Zero 플랜 무료 (월 설치 12,000건), Growth 플랜 $0.05~/설치 |
| **Firebase Analytics API** | 사용자 지표(DAU, 리텐션) 조회 | 무료 | GA4 기본 제공, BigQuery 연동 시 별도 |

### B. 설정만 등록 (API 호출 코드 미구현)

아래 서비스는 관리자 페이지에서 API 키 등록 UI만 존재하며, **실제 호출 코드가 아직 구현되지 않았습니다**.
향후 구현 시 예상되는 비용을 함께 안내합니다.

| 플랫폼 | 현재 상태 | 향후 예상 용도 | 예상 API 비용 |
|--------|----------|--------------|-------------|
| **Figma API** | 키 등록만 가능 | 디자인 템플릿 자동 합성, 광고 소재 이미지 생성 | 무료 (REST API 호출 무과금), 단 Figma 플랜 필요: Starter 무료 / Professional $15/편집자/월 / Organization $45/편집자/월 |
| **App Store Connect API** | 키 등록만 가능 | iOS 앱 메타데이터 조회, 스크린샷 업로드, 리뷰 모니터링 | 무료 (Apple Developer Program $99/년 필수) |
| **Google Play Developer API** | 키 등록만 가능 | Android 앱 정보 조회, 스토어 리스팅 관리, 리뷰 모니터링 | 무료 (Google Play Console 등록비 $25 일회성) |
